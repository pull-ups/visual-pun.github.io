Can visual language models resolve textual ambiguity with visual cues?
Let visual puns tell you!

Jiwan Chung, Seungwon Lim, Jaehyun Jeon, Seungbeen Lee, and Youngjae Yu
Yonsei University
EMNLP 2024
Arxiv Code 🤗 Dataset

What is multimodal literacy?

  • The ability to effectively integrate multimodal contexts
    to bridge informational gaps.
  • Unlike conventional Visual Question Answering,
    the informational gap is only indirectly provided.
  • A typical example is multimodal ambiguity resolution,
    in which multimodal inputs help disambiguating textual meaning.

Why is it hard to evaluate?

  • Eliminating textual bias is challenging.
  • i.e. We should make sure that the multimodal input is necessary!

Our Solution:

  • Puns have intrinsic ambiguity.
  • Let’ s design visual cues to explain each meaning.
  • UNPIE: a controlled benchmark for multimodal ambiguity resolution!

A. Data Collection

    1. We started from an existing text-only pun dataset.
    2. Then, we retrieve and manually select natural images
matching each meaning of the puns.
    3. Then, we use AI models to generate synthetic images
that explain both meanings of the puns at the same time.
▼ Show Details

Human annotators engaged in multi-turn editing with following criteria:
   1. The image should explain both meanings of the pun.
   2. The scene text should not directly explain the pun.

    4. Finally, we ask human translators to translate each
meaning of a pun into the target language (De, Fr, and Ko).







🤔 Why do we need translation?

  • Pun understanding is hard to evaluate.
  • We offer a workaround: composing the latent “understanding” with a downstream task (translation) enables tangible evaluation.
  • We can evaluate the translation task using existing VQA models.

B. Tasks and Experiments

Task 1: Pun Grounding

  • Specification:
    Given a pun and the dual meaning image,
    ground the pun phrase.
  • Purpose:
    Assess if visual context aids weak models
    in tasks solvable without the visuals.
  • Findings:
    Weaker models benefit a lot from visual context.

Results on the pun grounding task. We report the exact match accuracy of the generated pun phrase. ↑ denotes the performance gain from visual context.

Task 2: Pun Disambiguation

  • Specification:
    Given a pun and a single meaning image,
    translate to the target language while adhering to
    the given meaning.
  • Purpose:
    Controlled evaluation of whether the visual context actually affects model decisions.
  • Findings:
    Models integrate visual context, albeit at a limited degree.

Experimental results on the pun disambiguation task. All scores are reported in terms of binary classification accuracy. The best scores are bolded and the second-best ones are underlined.

Task 3: Pun Reconstruction

  • Specification:
    Given a translated pun and the dual meaning image, reconstruct the original pun sentence.
  • Purpose: Holistic downstream evaluation.
  • Findings: Stronger models benefit more from visual context.

Experimental results on the pun reconstruction task. We report the percentage of correct reconstructions, BLEU-4, and METEOR scores. The best scores are bolded and improvements are shown in parentheses.

Conclusion

  • Machines can indeed leverage visual information.
  • However, achieving human proficiency in multimodal literacy is still a challenge.
